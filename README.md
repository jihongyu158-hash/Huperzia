# Huperzia
# Huperzia plastome phylogenomics and spatiotemporal evolution

## Overview
This repository contains scripts used for the phylogenomic, biogeographic, diversification, spatial phylogenetic, and selection-pressure analyses of Chinese *Huperzia*. The workflow supports analyses described in the manuscript on the spatiotemporal evolution of Chinese *Huperzia*, including ancestral habitat reconstruction, diversification-rate estimation, spatial phylogenetic endemism and CANAPE analyses, morphological trait-associated diversification analyses, and branch-site tests of plastid protein-coding genes.
